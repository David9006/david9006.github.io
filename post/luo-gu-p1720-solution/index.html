洛谷 P1720 题解

2022-05-04

（题目链接：https://www.luogu.com.cn/problem/P1720）

1. 思路

稍加观察，你即可发现题目中的公式为斐波那契数列的通项公式（斐波那契数列：https://baike.baidu.com/item/%E6%96%90%E6%B3%A2%E9%82%A3%E5%A5%91%E6%95%B0%E5%88%97/99145），所以其实此题就是让你算斐波那契数列的第nnn项。

2. 代码

#include <bits/stdc++.h>
using namespace std;

long long fibs[50] = {1, 1};

void create_fibs() {
    for (int i = 2; i < 50; i++) {
         fibs[i] = fibs[i - 1] + fibs[i - 2];
    }
}

int main() {
    int n;
    cin >> n;

    create_fibs();

    cout << fibs[n - 1] << ".00";    // 注意，此题要求保留两位小数
    return 0; 
}
