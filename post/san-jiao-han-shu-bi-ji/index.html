定义

这里有两种定义方式：

  1. 如下图，对于 △ABC\triangle ABC△ABC，我们定义：

    • sin⁡α=c/a\sin \alpha = c/asinα=c/a（对边比斜边）.
    • cos⁡α=b/a\cos \alpha = b/acosα=b/a（邻边比斜边）.
    • tan⁡α=c/b\tan \alpha = c/btanα=c/b（对边比邻边）.
    • cot⁡α=b/c\cot \alpha = b/ccotα=b/c（邻边比对边）.
    • sec⁡α=a/c\sec \alpha = a/csecα=a/c（斜边比邻边）.
    • csc⁡α=a/b\csc \alpha = a/bcscα=a/b（斜边比对边）.

  2. 对于一个单位圆（其方程为 x2+y2=1x^2+y^2=1x2+y2=1），我们以 (1,0)(1, 0)(1,0) 为起点逆时针在圆上取一条长度为 α\alphaα 的弧，则其另一端点的坐标为 A(cos⁡α,sin⁡α)A(\cos \alpha, \sin \alpha)A(cosα,sinα). 同时，我们定义：

    • tan⁡α\tan \alphatanα 为直线 OAOAOA 的斜率.
    • cot⁡α=1tan⁡α\cot \alpha = \frac{1}{\tan \alpha}cotα=tanα1​.
    • sec⁡α=1sin⁡α\sec \alpha = \frac{1}{\sin \alpha}secα=sinα1​.
    • csc⁡α=1cos⁡α\csc \alpha = \frac{1}{\cos \alpha}cscα=cosα1​.

  1. sin⁡2α+cos⁡2α=1\sin^2 \alpha + \cos^2 \alpha = 1sin2α+cos2α=1.

    证明：在 Rt△ABC\text{Rt} \triangle ABCRt△ABC 中，设 α=∠ACB\alpha = \angle ACBα=∠ACB. 则 sin⁡2α=AB2/BC2,cos⁡2α=AC2/BC2\sin^2 \alpha = AB^2/BC^2, \cos^2 \alpha = AC^2/BC^2sin2α=AB2/BC2,cos2α=AC2/BC2. 则 sin⁡2α+cos⁡2α=AB2+AC2BC2=1\sin^2 \alpha + \cos^2 \alpha = \dfrac{AB^2 + AC^2}{BC^2} = 1sin2α+cos2α=BC2AB2+AC2​=1（此处 BCBCBC 为直角边）.

  2. 若 α+β=90°\alpha + \beta = 90 \degreeα+β=90°，则 sin⁡α=cos⁡β,cos⁡α=sin⁡β\sin \alpha = \cos \beta, \cos \alpha = \sin \betasinα=cosβ,cosα=sinβ.

  3. 1≤sin⁡α+cos⁡α<2.(0°≤α≤90°)1 \le \sin \alpha + \cos \alpha < 2. \quad (0 \degree \le \alpha \le 90 \degree)1≤sinα+cosα<2.(0°≤α≤90°)

    证明：

    • 首先证明 sin⁡α+cos⁡α≥1\sin \alpha + \cos \alpha \ge 1sinα+cosα≥1：因为 (sin⁡α+cos⁡α)2=1+2sin⁡αcos⁡α(\sin \alpha + \cos \alpha)^2 = 1 + 2 \sin \alpha \cos \alpha(sinα+cosα)2=1+2sinαcosα，且 sin⁡αcos⁡α≥0\sin \alpha \cos \alpha \ge 0sinαcosα≥0，所以 (sin⁡α+cos⁡α)2≥1(\sin \alpha + \cos \alpha)^2 \ge 1(sinα+cosα)2≥1，则 sin⁡α+cos⁡α≥1\sin \alpha + \cos \alpha \ge 1sinα+cosα≥1.
    • 然后证明 sin⁡α+cos⁡α<2\sin \alpha + \cos \alpha < 2sinα+cosα<2：若 sin⁡α+cos⁡α≥2\sin \alpha + \cos \alpha \ge 2sinα+cosα≥2，则 sin⁡α\sin \alphasinα 与 cos⁡α\cos \alphacosα 中至少有一个 ≥1\ge 1≥1，但这不可能，所以 sin⁡α+cos⁡α<2\sin \alpha + \cos \alpha < 2sinα+cosα<2.

    综上所述，1≤sin⁡α+cos⁡α<2.(0°≤α≤90°)1 \le \sin \alpha + \cos \alpha < 2. \quad (0 \degree \le \alpha \le 90 \degree)1≤sinα+cosα<2.(0°≤α≤90°)
