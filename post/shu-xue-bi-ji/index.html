1. 对于实数 a,ba, ba,b，有 ∣a∣+∣b∣≥∣a+b∣≥∣∣a∣−∣b∣∣|a| + |b| \ge |a + b| \ge ||a| - |b||∣a∣+∣b∣≥∣a+b∣≥∣∣a∣−∣b∣∣. 其中在 a,ba, ba,b 同号时第一个不等号取等，在 a,ba, ba,b 异号时第二个不等号取等.

证明：设 a>ba > ba>b，考虑 a,ba, ba,b 的符号：

  • a,ba, ba,b 同号：∣a∣+∣b∣=∣a+b∣|a| + |b| = |a + b|∣a∣+∣b∣=∣a+b∣，∣a+b∣>∣∣a∣−∣b∣∣|a + b| > ||a| - |b||∣a+b∣>∣∣a∣−∣b∣∣.
  • a,ba, ba,b 异号：∣a∣+∣b∣=b−a=∣a−b∣>∣a+b∣|a| + |b| = b - a = |a - b| > |a + b|∣a∣+∣b∣=b−a=∣a−b∣>∣a+b∣，∣a+b∣=∣a∣−∣b∣=∣∣a∣−∣b∣∣|a + b| = |a| - |b| = ||a| - |b||∣a+b∣=∣a∣−∣b∣=∣∣a∣−∣b∣∣.

原命题得证.

思路：

Q1. 如何用加法表示两数同号？
A1. ∣a∣+∣b∣=∣a+b∣|a| + |b| = |a + b|∣a∣+∣b∣=∣a+b∣ 时 a,ba, ba,b 同号.

Q2. 在其余情况下不等式的不等号是什么？
A2. 其余情况下 ∣a∣+∣b∣>∣a+b∣|a| + |b| > |a + b|∣a∣+∣b∣>∣a+b∣.

Q3. 能在现有不等式前后加什么东西？
A3. 可以在不等式后加 ∣∣a∣−∣b∣∣||a| - |b||∣∣a∣−∣b∣∣，当前不等式变为 ∣a∣+∣b∣≥∣a+b∣≥∣∣a∣−∣b∣∣|a| + |b| \ge |a + b| \ge ||a| - |b||∣a∣+∣b∣≥∣a+b∣≥∣∣a∣−∣b∣∣.

推广：

对于两个向量 a⃗,b⃗\vec{a}, \vec{b}a,b，有 ∣a⃗∣+∣b⃗∣≥∣a⃗+b⃗∣|\vec{a}| + |\vec{b}| \ge |\vec{a} + \vec{b}|∣a∣+∣b∣≥∣a+b∣. 考虑向量 a⃗,b⃗,a⃗+b⃗\vec{a}, \vec{b}, \vec{a} + \vec{b}a,b,a+b 有没有组成三角形：

  • 如果有，则 ∣a⃗∣+∣b⃗∣≥∣a⃗+b⃗∣|\vec{a}| + |\vec{b}| \ge |\vec{a} + \vec{b}|∣a∣+∣b∣≥∣a+b∣（三角形的两边之和大于第三边）.
  • 如果没有，则 ∣a⃗∣+∣b⃗∣=∣a⃗+b⃗∣|\vec{a}| + |\vec{b}| = |\vec{a} + \vec{b}|∣a∣+∣b∣=∣a+b∣.

故 ∣a⃗∣+∣b⃗∣≥∣a⃗+b⃗∣|\vec{a}| + |\vec{b}| \ge |\vec{a} + \vec{b}|∣a∣+∣b∣≥∣a+b∣.
