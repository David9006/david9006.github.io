Codeforces 29 A 题解

2022-05-05 Codeforces

（题目链接：https://codeforces.com/problemset/problem/29/A）
（我的提交：https://codeforces.com/contest/29/submission/155919409）

1. 题意

在一个动物园里，有很多骆驼。众所周知，骆驼喜欢吐口水。Bob花了一整天看他们吐口水，并且在他的笔记本上记下了每只骆驼在吐口水时所在的位置。现在，他想知道有没有一对骆驼互相向对方吐口水。

如果一只骆驼在 x(−104≤x≤104)x (-10^4 \le x \le 10^4)x(−104≤x≤104) 位置吐出了 d(1≤∣d∣≤2×104)d (1 \le |d| \le 2 \times 10^4)d(1≤∣d∣≤2×104) 长度的口水，则吐出的口水只会打到 (x+d)(x + d)(x+d) 位置的骆驼。

2. 思路

  • 首先，我们用一个结构体或一个std::pair把所有骆驼存下来；
  • 用一个双重循环遍历所有骆驼，把正在遍历的两个骆驼进行判断：它们两个是否在互吐口水；
  • 按照第二步的判断结果来相应输出YES或NO。

3. 代码

// Codeforces Beta Round #29 (Div. 2, Codeforces format) - A. Spit Problem
#include <iostream>
#include <vector>

#define OPTIMIZE() do {             \
    ios::sync_with_stdio(false);    \
    cin.tie(nullptr);               \
    cout.tie(nullptr);              \
} while (0);

using namespace std;

struct Camel {
    int pos, spit_len;
};

int main() {
    OPTIMIZE();

    int n;
    vector<Camel> camels;

    cin >> n;
    camels.resize(n);

    for (auto& camel : camels) {
        cin >> camel.pos >> camel.spit_len;
    }

    // 用双重循环挨个遍历所有骆驼
    for (int i = 0; i < n; ++i) {
        for (int j = i + 1; j < n; ++j) {
            if (camels[i].pos + camels[i].spit_len == camels[j].pos && camels[j].pos + camels[j].spit_len == camels[i].pos) {
                cout << "YES" << endl;
                return 0;    // 只要找到答案，就立刻退出程序，不然还会输出一遍"NO"
            }
        }
    }
    
    cout << "NO" << endl;
    return 0;
}
