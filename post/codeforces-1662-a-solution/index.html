Codeforces 1662 A 题解

2022-05-04 Codeforces

1. 题意

Gianni收到了nnn道题目，每道题目都有难度di(1≤di≤10)d_i (1 \le d_i \le 10)di​(1≤di​≤10)和美丽值bi(1≤bi≤10)b_i (1 \le b_i \le 10)bi​(1≤bi​≤10)。如果没有一道题对应着难度ddd，Gianni将会继续找同事索要题目。否则，Gianni会在每一个难度中挑选美丽值最大的题，然后把挑选出来的题编成习题集。如果Gianni需要找同事索要题目，你需输出MOREPROBLEMS，否则输出习题集里所有的题的美丽值的总和。

2. 思路

  • 维护一个数组，数组的下标为难度，数组的元素代表其对应的难度中所有题目的美丽值的最大值，随后把此数组所有元素初始化为000；
  • 在输入时，计算出对于每个难度中所有题目的美丽值的最大值；
  • 如果数组中有元素为0，说明那个元素所对应的难度中没有题目，输出MOREPROBLEMS；
  • 否则，输出此数组所有元素的总和。

3. 代码

// SWERC 2021-2022 - Online Mirror (Unrated, ICPC Rules, Teams Preferred) - A. Organizing SWERC
#include <cstring>    // for memset()
#include <iostream>
#include <numeric>    // for accumulate()

#define OPTIMIZE() do {             \
    ios::sync_with_stdio(false);    \
    cin.tie(nullptr);               \
    cout.tie(nullptr);              \
} while (0);

using namespace std;

void solve() {
    int n;
    int problems[15];    // 存储习题集，元素为15是为了防止溢出

    cin >> n;
    memset(problems, 0, sizeof(problems));

    for (int i = 0; i < n; i++) {
        int b, d;
        cin >> b >> d;

        problems[d] = max(problems[d], b);    // 更新难度d代表的所有题目的美丽值的最大值
    }
    
    // 判断是否需要索要题目
    if (n < 10) {
        cout << "MOREPROBLEMS" << endl;
        return;
    }

    for (int i = 1; i <= 10; i++) {
        if (problems[i] == 0) {
            cout << "MOREPROBLEMS" << endl;
            return;
        }
    }

    cout << accumulate(problems + 1, problems + 11, 0) << endl;    // 求出习题集中所有题的总和
}

int main() {
    OPTIMIZE();

    int T;
    cin >> T;

    while (T--) {
        solve();
    }
    
    return 0;
}
